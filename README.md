# Responsive Digital Invitation 🎂

An elegant, mobile-first web invitation designed with modern UI trends and interactive features.

### Key Features:
* **Interactive Countdown:** A real-time JavaScript countdown timer tracking the event date.
* **Celebratory Effects:** Integrated `Canvas-Confetti API` for a premium user entrance experience.
* **Glassmorphism Design:** Uses advanced CSS filters and mesh gradients for a sophisticated look.
* **Multilingual:** Supports both English and Tamil typography.

### Tech Stack:
* HTML5, CSS3, JavaScript, Canvas-Confetti API, Google Fonts.
